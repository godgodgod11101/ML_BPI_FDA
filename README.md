# ML_BPI_FDA
邊境智能預測(Border Predict Intelligence, BPI)模型調整測試。  

針對**農產品類**進口報單模型進行資料調整評估，包含：  
- 調整**資料擷取期間**：參考**農藥殘留法規變動**時間點，擷取不同期間資料建模，評估是否能提升模型預測表現。  
- 放入**高風險特徵**：設計不同高風險特徵放入模型，評估是否能提升模型預測表現。
